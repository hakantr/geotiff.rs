# README raster attribution

`mount-yasur-sar.png` is generated by this repository's
`examples/render_png.rs` from `umbra_mount_yasur.tiff` at the pinned
`GeoTIFF/test-data` commit
`8506204783ff26a6c49ed1f721e7e1635b2e43ee`.

- Subject: Mount Yasur, Vanuatu, synthetic-aperture radar image
- Source and creator: [Umbra Space Open Data Program](https://umbra.space/open-data)
- Source fixture: [GeoTIFF/test-data](https://github.com/GeoTIFF/test-data/blob/8506204783ff26a6c49ed1f721e7e1635b2e43ee/files/umbra_mount_yasur.tiff)
- Image license: [Creative Commons Attribution 4.0 International](https://creativecommons.org/licenses/by/4.0/)
- Changes: decoded from the 2000 x 2000 BlackIsZero GeoTIFF through
  `geotiff.rs`, bilinearly resampled to 800 x 800 through `read_rgb`, and
  serialized as a grayscale PNG. No other visual alteration was applied.

The generated PNG is reopened by the example, expanded to RGB, and compared
pixel for pixel with the port's decoded and resampled output before the example
reports success.
